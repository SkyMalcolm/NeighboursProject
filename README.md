# Neighbours Project

Initial userflow

## Lathund git

#### Skapa branch
```
git status
git branch
git checkout -b branchname
```

#### Delete branch
```
git checkout main
git pull
git branch -d branchname
```

#### Get new changes from main to branch 
```
git checkout branchname
git merge origin main
```

#### Commit
```
git diff main (för att kolla vad som ändras)
git add .
git commit -m ”commit”
```

#### Push
```
git push 
git push —set.upstream osv
```

#### Pull
```
git pull
```
